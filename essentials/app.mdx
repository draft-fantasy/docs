## Installation

To install the Draft Fantasy app, visit [app.draftfantasy.com](https://app.draftfantasy.com) on your phone.
Then click `Share` and then click `Add to Homescreen`.

### How to install on Chrome iOS:

The steps are the same for Safari iOS, but you'll find the `Share` icon button in the bottom center of the screen.

Click the `Share` icon button in the top right corner:

### Install on Desktop

You can also install the app on your desktop by visiting [app.draftfantasy.com](https://app.draftfantasy.com) on your desktop device.
You will see an install icon at the end of the URL bar. Click it to install the app.
